What is base64?

Base64 is group of ASCII characters that represent to binary data. We present audio, video or any file in text format. The term base64 originates from a specific MIME content transfer encoding

What is base64 used for?

Most commonly used cases of Base64 is help us to sent file with in multiple format through a system which only supports for text data or ASCII text. You can see the usage of base64 when inspect web element for img tag in some websites.

With base64, we can easily embedded images or videos in HTML without file-server to host those files.
